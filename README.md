This application is a landing page to Aker.
Users can get to other Aker applications from links in the navbar.

# Setup

## Dev environment
To install gems, run:
`bundle install`
Configure ports to services in `development.rb`.

# Testing
To run all test for this project, execute:
`bundle exec rake`
